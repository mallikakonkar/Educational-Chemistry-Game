INSTRUCTIONS

OBJECTIVE

To guess the correct properties for all the elements in the periodic table!

METHOD

For every element select the property that you think is correct if the property is correct the ball will move towards the pole and if its incorrect it will move towards the pond.

RULES

  1. A question is marked as correct if the user has guessed 3 correct answers

  2. A question is marked as incorrect if the user has guessed 2 incorrect answers
